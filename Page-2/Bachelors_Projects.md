# Toy Model to Explain Superconductivity

- Charges with the same polarity typically repel each other. However, in superconductors, electrons can form bound pairs known as *Cooper pairs*.

- The BCS theory explains this phenomenon using a quantum mechanical description of electron–phonon interactions.

- In this project, I developed a simple toy model to demonstrate how two repelling electrons can experience an effective attractive interaction mediated by lattice vibrations.

- The work was designed as an intuitive and pedagogical approach to understanding the microscopic origin of Cooper pair formation using classical mechanics concepts.

- The simulation code and implementation have been made openly available on GitHub.
